🤖 Telegram-бот

Интеллектуальный бот для записи клиентов — работает 24/7 без выходных и праздников.

⚡
Запись через 3 клика
Клиент выбирает услугу → дату → время. Готово! Среднее время записи — 15 секунд.
📅
Управление записями
Просмотр активных записей, изменение даты/времени, отмена — всё внутри бота.
🔔
Автонапоминания
Бот отправляет напоминания за 24 часа и за 2 часа до записи. Забывших клиентов больше нет.
🧠
FSM на 5 состояний
Конечный автомат обеспечивает чёткую логику: выбор услуги → дата → время → подтверждение → готово.
📊
Админ-панель
Просмотр всех записей на сегодня/завтра/неделю. Управление расписанием прямо в Telegram.
🌐
Без внешних библиотек
Самописный календарь без сторонних зависимостей. Никаких подписок, только ваш сервер.

🖥️ Backend

Надёжная серверная часть на Node.js с Google Sheets в качестве базы данных.

🚀
Express.js REST API
Лёгкий и быстрый API для приёма заявок с лендинга, проверки слотов и синхронизации с ботом.
📋
Google Sheets как БД
Все записи хранятся в Google Таблице. Удобно смотреть, фильтровать, выгружать в CRM.
📬
Уведомления админу
Каждая новая запись приходит в Telegram. Мгновенно видно нового клиента.
🛡️
Rate-limit + валидация
Защита от спама, проверка телефона, валидация всех входящих данных.

🌐 Лендинг

Продающий лендинг с калькулятором тарифов для привлечения клиентов.

💰
Калькулятор тарифов
3 тарифа на выбор. Клиент считает стоимость сам — прозрачно и без менеджера.
📱
Mobile-first дизайн
Адаптивная вёрстка под все устройства. Отлично выглядит на телефоне.
✨
Анимации и частицы
Canvas-анимация на фоне, плавные переходы, премиум-ощущение.
📋
Форма заявки
Простая форма с валидацией. Заявка уходит в бот и Google Sheets.
